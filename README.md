# AI Symptom Guidance Assistant

An LLM-powered assistant that helps users describe their symptoms in natural language and receive preliminary guidance on possible causes and recommended next steps. Built as part of an AI Engineering curriculum, using the AfyaPlus healthcare scenario as a design context.

## Overview

This project explores how large language models can be applied to healthcare guidance use cases — taking unstructured symptom descriptions as input and producing clear, actionable, and appropriately cautious output for users.

**Important:** This tool is for educational/demonstration purposes only and is not a substitute for professional medical advice, diagnosis, or treatment.

## Features

- Natural language symptom input
- LLM-powered analysis and guidance generation
- Recommendations on next steps (e.g., self-care, seek care, urgent care)
- Built with safety and responsible AI guardrails in mind
